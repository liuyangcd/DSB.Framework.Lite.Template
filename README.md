# DSB.Framework.Lite.Template

基于 `DSB.Framework.Lite` 框架实现的 ASP.NET Core 8 解决方案模板。

本框架的主要目标是帮助开发者实现中小型项目的**快速、简单、高效**开发和迭代。

## 核心功能

- Swagger 文档优化：支持 API 分组配置和注释优化，提升接口文档的可读性和管理效率。
- 任务调度集成：内置 Hangfire，轻松实现后台任务、定时任务和周期性任务的调度与管理。
- 分布式锁：基于 Redis 实现业务自定义范围控制的分布式锁，有效解决高并发场景下的资源竞争问题。
- 文件上传管理：提供一套快速配置和管理的文件上传系统，简化文件处理流程。
- 可配置加密接口：实现可配置的接口加密设计，灵活保障数据传输安全。
- 自定义仓储模式：实现自定义数据库仓储（Repository）模式，提供更优雅和统一的数据访问层抽象。
- 网络访问扩展：通过配置文件可快速配置第三方接口访问。
- 安全模块：
    - 实现基于 PBKDF2 算法的密码管理，增强账户安全性。
    - 内置基础图形验证码模块，有效防止恶意请求。
- 身份与权限管理：
    - 实现基于 JWT (JSON Web Token) 的身份验证。
    - 支持自定义权限码模式的权限管理，满足灵活的授权需求。

本项目只作为项目模板示例作用；细节直接查看代码。
